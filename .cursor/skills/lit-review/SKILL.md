# Literature Review Skill

## Overview

Search, summarize, and synthesize economics literature for research papers.

## Steps

### Phase 1: Scope Definition
1. Clarify the research question
2. Identify key concepts and terms
3. Define inclusion/exclusion criteria
4. Set scope boundaries

### Phase 2: Search Strategy
1. Identify core databases:
   - NBER Working Papers
   - SSRN
   - Google Scholar
   - EconLit
   - RePEc
2. Develop search queries
3. Use citation chaining (forward and backward)

### Phase 3: Paper Collection
1. Screen titles and abstracts
2. Retrieve full texts
3. Organize by theme/methodology
4. Track in reference manager

### Phase 4: Synthesis
1. Identify main findings
2. Map methodological approaches
3. Find gaps in literature
4. Note contradictions/debates

### Phase 5: Output
1. Create structured summary
2. Generate citation network (if needed)
3. Identify seminal papers
4. Recommend positioning
